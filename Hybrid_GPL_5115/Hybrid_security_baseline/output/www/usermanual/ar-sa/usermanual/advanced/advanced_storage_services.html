تهيئة خدمات التخزين

بفضل خدمات التخزين، يمكنك عرض الملفات المشتركة الموجودة على جهاز USB أو تعديلها أو إضافتها أو حذفها.

شرط مبدئي

قبل إجراء هذه العملية، يُرجى التأكد من توصيل جهاز USB بـ . يمكن لـ التعرف على 11 قسمًا منطقيًا كحدٍ أقصى.

الإجراء

  1. اختر < < .
  2. حدد أو أو كليهما لتمكين خدمات التخزين. انقر فوق .
  3. انقر فوق لإضافة مسار مشترك وتهيئة المعلمات. للحصول على مزيدٍ من التفاصيل حول المعلمات، انظر .
  4. انقر .

    ملاحظة:

    يمكنك تحرير عنصر موجود أو حذفه عن طريق النقر فوق الزر أو الزر الموجود خلفه.

شروط لاحقة

ملاحظة:
  • يُستخدم جهاز كمبيوتر يعمل بنظام التشغيل Windows 7 كمثال في العملية التالية.
  • يُرجى عدم فصل جهاز USB من مباشرة عقب إضافة ملف (ملفات) أو تعديله أو حذفه، ولكن انتظر لمدة 5 ثوانٍ حتى يتم تفعيل العملية.
  • إذا لم تشاهد تأثير جهاز USB بعد إضافة الملفات أو تعديلها أو حذفها، يجب تحديث المجلد يدويًّا (انقر بزر الماوس الأيمن فوق الفراغ الموجود في المجلد، ثم اختر تحديث، لرؤية التأثيرات التي تم تفعيلها.
  • في حال استخدام جهاز كمبيوتر يعمل بنظام التشغيل Windows XP، يُرجى الوصول إلى جهاز USB باستخدام عنوان IPv4، إذ إن عنوان IPv6 غير مدعوم بواسطة Windows XP؛ بالتالي قد يتعذر عليك الوصول إلى عنوان IPv6.

عرض الملف المشترك باستخدام خادم سامبا

  1. قم بتشغيل جهاز الكمبيوتر.
  2. قم بتوصيل الكمبيوتر بـ عبر Ethernet أو Wi-Fi.
  3. بدء التشغيل < تشغيل وأدخل عنوان IP الخاص بـ (عنوان IPv4 الافتراضي: ‎\\192.168.1.1 وعنوان IPv6 الافتراضي: ‎\\fe80--1.ipv6-literal.net)، ثم اضغط إدخال.
  4. أدخل اسم المستخدم وكلمة المرور اللذين تم تهيئتهما في صفحة .
  5. قم بعرض الملفات المشتركة.

عرض الملف المشترك باستخدام خادم FTP

  1. قم بتشغيل جهاز الكمبيوتر.
  2. قم بتوصيل الكمبيوتر بـ عبر Ethernet أو Wi-Fi.
  3. قم بتشغيل مستعرض ويب، ثم أدخل ftp://192.168.1.1 (عنوان IPv4) أو ftp://[fe80::1]‎ (عنوان IPv6) لتسجيل الدخول إلى الإدارة.
  4. أدخل اسم المستخدم وكلمة المرور اللذين تم تهيئتهما في صفحة .
  5. قم بعرض الملفات المشتركة.
