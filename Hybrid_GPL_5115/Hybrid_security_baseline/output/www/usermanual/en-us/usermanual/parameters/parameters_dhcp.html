DHCP

  • DHCP
    The automatically allocates network parameters, such as an IPv4 address and DNS server address, to your computer, and implements centralized management of IPv4 addresses.
  • IPv6 DHCP
    If your computer support IPv6, you can use IPv6 DHCP server to automatically allocates network parameters, such as an IPv6 address and DNS server address.
  • SLAAC
    After IPv6 DHCP server is enabled, you can use Stateless Address Auto-Configuration (SLAAC) to automatically configure IPv6 parameters for your computer.
  • Static Address
    With static address, you can assign static LAN IPv4 address for a specific computer.
