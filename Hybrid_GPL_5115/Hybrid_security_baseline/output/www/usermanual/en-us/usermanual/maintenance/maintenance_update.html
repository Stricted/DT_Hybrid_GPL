Updating the Device

After your is connected to the Internet, you can use this function to download the upgrade package from Huawei's cloud server and upgrade the .

Prerequisite

WARNING:

Ensure that the is connected to the Internet before updates. when upgrade is processing, do not close the web browser or power off the .

Procedure

  1. Select > .
  2. Click to check for updates.
  3. If there is a new version available, click to download upgrade package.
  4. After download is complete, click to start upgrade.
  5. Wait until the restart.
