Sharing Files on USB Storage Device

With USB storage services, you can view, modify, add or delete the shared file(s) on the USB device.

Prerequisite

Before this operation, please ensure that a USB device has been connect to . The can recognize a maximum of 11 logical partitions.

Procedure

  1. Select > > .
  2. Check one or both of the and to enable Storage Services. Click .
  3. Click to add a shared path and configure the parameters. For details about the parameters, see Storage Services.
  4. Click .

    NOTE:

    You can edit or delete an existed item by click the or button behind it.

Postrequisites

NOTE:
  • The following operation are taking a computer with Windows 7 operating system as an example.
  • Please do NOT disconnect the USB device to the IMMEDIATELY when you just have add, modify or delete file(s), Wait 5 seconds for the operation to take effect.
  • If you can not see the effect of the USB device after you add, modify or delete file(s), you should manually refresh the folder (right click on the blank of the folder and choose refresh), to see the effect that have been taken.
  • If you are using a computer with Windows XP operating system, please access the USB device by IPv4 address. The IPv6 is not well supported by Windows XP, you may failed to access by IPv6 address.

View the shared file by Samba.

  1. Power on the computer.
  2. Connect the computer to via wired way or Wi-Fi way.
  3. Choose Start > Run, enter the IP address of the (default IPv4 address: \\192.168.1.1, default IPv6 address: \\[fe80::1]), and then press Enter.
  4. Enter the user name and password you have configured in the page.
  5. View the shared files.

View the shared file by FTP Server.

  1. Power on the computer.
  2. Connect the computer to via wired way or Wi-Fi way.
  3. Launch a web browser and enter ftp://192.168.1.1 (IPv4) or ftp://[fe80::1] (IPv6) to log in to the management page.
  4. Enter the user name and password you have configured in the page.
  5. View the shared files.
