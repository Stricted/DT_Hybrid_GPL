Why am I unable to access WLAN on a client?

Procedure

  1. Interferences or obstacles can block wireless signal. Move the to a different location.
  2. Check and record the settings of the following parameters on the clients and the : , security key and . The service set identifier (SSID) of the clients must be set to ANY or be same as that set on the . The encryption mode and the security key on the clients must match those set on the . Otherwise, change the settings on the clients.
  3. If the Media Access Control (MAC) address filtering in the wireless local area network (WLAN) is enabled on the , check that the MAC addresses of the clients are not in the list. For details, see LAN MAC Filter.
