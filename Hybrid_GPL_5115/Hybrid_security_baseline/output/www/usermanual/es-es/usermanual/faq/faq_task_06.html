How to add a printer?

NOTE:
  • Take a computer with Windows 7 operating system as an example.
  • If the external USB printer cannot be used, please consult the printer manufacturer whether this method is supported.
  1. Install the printer's driver in your computer, and then connect the computer and via wired way or Wi-Fi way.
  2. Connect printer to with USB cable and then power on them.
  3. Log in to web management page, and select > > .
  4. Check the , and click .

  5. Click , to configure the parameters as follows.

    • /: Follow the page to enter username and password. When accessing Samba, you need to input these parameters to login.
    • : Select or .
    • : Click and select the root directory of the printer.
  6. Choose Start > Run, enter the default IP address of the : \\192.168.1.1, and then press Enter.
  7. In the shown dialog box, enter the username and password you have configured on the page to access to the Samba service.
  8. Double click the printer icon in the window, follow the wizard and select the corresponding printer driver to add a printer. Wait a few minutes for the settings to take affect.
  9. Check the status of printer from Control Panel, printer window. Status should be shown ready. Now you can use the printer.
