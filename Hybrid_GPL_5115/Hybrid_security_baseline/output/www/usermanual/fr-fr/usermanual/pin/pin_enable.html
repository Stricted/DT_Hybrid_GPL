Enabling PIN Code Verification

Enable PIN code verification, prevent unauthorized users from using your SIM/USIM card. You need to validate the PIN at each startup of the to use the data or voice service through your SIM/USIM card.

Context

  • If you enter your PIN incorrectly three consecutive times, the SIM/USIM card will be locked and you must enter the PIN Unlock Key (PUK).
  • If you enter the PUK incorrectly ten consecutive times, the SIM/USIM card will be locked permanently. If you cannot enter the correct PIN or PUK, network-associated functions are unavailable.
  • The PIN and the PUK are provided with the SIM/USIM card. If you did not receive them or forgot them, contact your service provider.
