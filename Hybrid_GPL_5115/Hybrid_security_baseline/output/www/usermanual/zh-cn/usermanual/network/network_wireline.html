新建有线互联网配置文件

您可以根据不同的有线联网环境，分别建立对应的配置文件。当您的有线联网环境发生变化时，通过在“”中切换配置文件即可更新联网参数，同时您原先的联网参数也得以保留。

操作步骤

  1. 选择“ > > ”。
  2. 单击“”。
  3. 设置相关参数。具体参数解释请参见有线 WAN 配置文件。
  4. 单击“”。

    说明：

    您也可以从“”中选择一条配置文件，单击“”或“”按钮编辑或删除一条已有的配置文件。
