互联网连接设置

  • 新建无线互联网配置文件
    您可以根据不同的无线（4G/3G/2G）联网环境，分别建立对应的配置文件。当您的无线联网环境发生变化时，通过在“”中切换配置文件即可更新联网参数，同时您原先的联网参数也得以保留。
  • 新建有线互联网配置文件
    您可以根据不同的有线联网环境，分别建立对应的配置文件。当您的有线联网环境发生变化时，通过在“”中切换配置文件即可更新联网参数，同时您原先的联网参数也得以保留。
  • 通过 WPS PIN 方式建立无线连接
    WPS（WLAN Protected Setup）是一种用于简化无线网络安全设置的标准。通过使用 WPS 功能，您无需输入无线网络名称和密码就可以使无线终端安全地加入到无线网络。
