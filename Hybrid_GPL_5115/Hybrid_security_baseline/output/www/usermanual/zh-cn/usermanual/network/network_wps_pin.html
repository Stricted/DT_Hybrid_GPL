通过 WPS PIN 方式建立无线连接

WPS（WLAN Protected Setup）是一种用于简化无线网络安全设置的标准。通过使用 WPS 功能，您无需输入无线网络名称和密码就可以使无线终端安全地加入到无线网络。

背景信息

说明：

  1. 选择“ > > ”。
  2. 从“”中，选择一个 WPS PIN 模式。并开启无线客户端的 WPS PIN 功能。
    • 选择“”，您需要在“”中输入无线客户端上显示的 WPS PIN 码。
    • 选择“”，您需要在无线客户端的 WPS PIN 输入框中输入“”显示的 WPS PIN 码。
      说明：

      无线客户端 WPS PIN 功能的操作可能因设备不同而有所不同，具体操作请查阅该设备的相关说明书。

  3. 单击“”。
