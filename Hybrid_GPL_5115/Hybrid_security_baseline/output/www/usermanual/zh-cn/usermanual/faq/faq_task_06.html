如何添加打印机？

说明：
  • 以下操作以 Windows 7 操作系统为例。
  • 如果打印机通过 USB 连接始终无法连接成功，请咨询您的打印机供应商该打印机是否支持 USB 连接模式。
  1. 在局域网中的某一台计算机中安装打印机驱动程序，然后将该计算机通过有线或无线方式连接到“”。
  2. 通过 USB 连接线分别连接“”和打印机，并开启打印机。
  3. 登录 Web 管理网页，并选择“ > > ”。
  4. 选择“”选项，并点击“”按钮。

  5. 单击“”按钮，设置相关参数。

    • “”和“”：输入登录 Samba 服务的自定义用户名和密码。
    • “”：选择“”或“”，指定用户的操作权限。
    • “”：单击“”按钮，选择到打印机的根目录。
  6. 在计算机中，选择“开始 > 运行”，输入“”的默认 IP 地址：\\192.168.1.1，按“回车”键。
  7. 在弹出的对话框中，输入您刚刚在“”中设置的用户名和密码登录。
  8. 在窗口中双击打印机图标，跟随添加想到选择对应的打印机驱动程序添加打印机。等待几分钟，以使设置生效。
  9. 在“控制面板”中，进入打印机窗口查看打印机是否可用。
