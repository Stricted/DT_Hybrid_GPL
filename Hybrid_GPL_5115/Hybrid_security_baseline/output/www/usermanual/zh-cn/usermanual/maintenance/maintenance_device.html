设备维护

  • 修改登录密码
    您在登录 Web 配置界面时，需要输入正确的用户名和密码。经常修改登录 Web 配置界面的用户名和密码，可以防止未经授权的人员随意修改 Web 配置界面的相关参数。
  • 恢复出厂设置
    如果您忘记了配置的 web 页面登录密码或无线局域网（WLAN）密码等自定义信息，或者“”运行出现不可恢复的异常时，您可以将“”恢复出厂设置。
  • 重新启动设备
    当“”出现运行缓慢或无线局域网（WLAN）连接不上等异常时，您可以尝试重新启动设备。
  • 查看日志信息
    您可以查看系统记录的各个等级日志的详细信息。
  • 备份设备配置
    您可以将当前已设置好的“”配置参数作为文件备份出来，以便在后续配置参数丢失时通过导入备份文件的方式，快速恢复之前的配置。
  • 升级设备
    在线升级功能可以使连接到互联网的“”，通过连接华为云服务器获取最新软件，并实现软件自动更新。
