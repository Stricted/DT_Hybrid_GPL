修改登录密码

您在登录 Web 配置界面时，需要输入正确的用户名和密码。经常修改登录 Web 配置界面的用户名和密码，可以防止未经授权的人员随意修改 Web 配置界面的相关参数。

操作步骤

  1. 选择“ > > ”。
  2. 在“”中，输入当前密码。
  3. 在“”中，输入新密码。
  4. 在“”中重复输入一遍新密码。
  5. 单击“”。

    说明：

    为了保护您的账号安全，请按照页面提示规则设置新密码。
