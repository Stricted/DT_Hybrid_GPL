通过 USB 存储设备共享文件

开启存储共享服务，使家庭网络中的多台计算机能通过局域网同时访问 USB 存储设备中的数据，并对共享的数据进行查看、修改、删除等操作。

前提条件

请在设置前，确认 USB 存储设备已经连接到“”。“”最多可以识别11个逻辑分区。

操作步骤

  1. 选择“ > > ”。
  2. 同时选择“”和“”或选择其一，开启存储共享服务，单击“”。
  3. 单击“”添加一条共享路径。具体参数设置请参见存储服务。
  4. 单击“”。

    说明：

    您可以通过点击“”或“”按钮，对一条已有的共享路径进行编辑或者删除。

后续处理

说明：
  • 以下操作以 Windows 7 操作系统为例。
  • 在对共享路径中的文件进行编辑、删除或增加操作后，请不要立即将 USB 存储设备从“”的 USB 口中拔出，等待5秒钟后再拔出。
  • 如果您在对共享文件夹中的文件进行编辑、删除或增加操作后，没有在文件夹中看到操作结果，请手动刷新一下文件夹。
  • 由于 Windows XP 操作系统不支持 IPv6 网络地址，当使用 Windows XP 操作系统访问共享存储路径时，请使用 IPv4 网络地址连接。

通过 Samba 方式访问共享路径

  1. 开启计算机。
  2. 将计算机通过有线或无线的方式连接到“”。
  3. 选择“开始 > 运行”，输入“”的局域网地址（默认 IPv4 地址为：\\192.168.1.1，默认 IPv6 地址为：\\fe80--1.ipv6-literal.net），按“回车”键。
  4. 连接成功后，在弹出的对话框中输入您在“”中设置的登录用户名和密码。
  5. 浏览共享文件。

通过 FTP 服务器方式访问共享路径

  1. 开启计算机。
  2. 将计算机通过有线或无线的方式连接到“”。
  3. 打开浏览器，在浏览器的地址栏中输入ftp://192.168.1.1（IPv4）或ftp://[fe80::1] （IPv6），按“回车”键。
  4. 连接成功后，在弹出的对话框中输入您在“”中设置的登录用户名和密码。
  5. 浏览共享文件。
