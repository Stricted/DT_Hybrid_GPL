高级设置

  • 通过 USB 存储设备共享文件
    开启存储共享服务，使家庭网络中的多台计算机能通过局域网同时访问 USB 存储设备中的数据，并对共享的数据进行查看、修改、删除等操作。
  • 播放 USB 存储设备中的多媒体文件
    通过 DMS（Digital Media Server，数字媒体服务器），连接到局域网中的计算机可以通过 “Windows Media Player”播放 USB 存储设备中的多媒体文件。
  • 修改来电显示模式
    如果您电话的来电显示出现异常，可以通过修改来电显示模式尝试恢复正常。
