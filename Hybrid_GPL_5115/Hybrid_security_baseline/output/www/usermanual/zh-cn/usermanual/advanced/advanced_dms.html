播放 USB 存储设备中的多媒体文件

通过 DMS（Digital Media Server，数字媒体服务器），连接到局域网中的计算机可以通过 “Windows Media Player”播放 USB 存储设备中的多媒体文件。

前提条件

请在设置前，确认 USB 存储设备已经连接到“”。

操作步骤

  1. 选择“ > > ”。
  2. 选择“”，开启 DMS。
  3. 单击“”，选择共享目录。

    说明：
    • 如果目录名称不能正常显示，清除选择“”。
    • 目录路径最多可包含4096个字符。被共享的目录中最多可以包含5000层子目录。

  4. 单击“”。

后续处理

播放共享文件（以 Windows 7 操作系统为例）

  1. 开启计算机。
  2. 将计算机通过有线或无线的方式连接到“”。
  3. 打开软件。
  4. 单击。
  5. 浏览共享文件。
