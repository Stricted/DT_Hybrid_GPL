开启 PIN 码验证

开启 PIN 码验证，可以有效避免非授权用户使用您的 SIM/USIM 卡。您需要在每次启动“”时验证 PIN 码，以通过 SIM/USIM 卡使用数据和通话业务。

背景信息

  • 如果您输入 PIN 码的错误次数超过3次，SIM/USIM 卡会被锁住。此时，您需要输入 PUK（PIN Unlocking Key，PIN码解锁密码）码解锁。
  • 如果您输入 PUK 码的错误次数超过10次，SIM/USIM 卡会被作废。此时，请联系您的 USIM 卡提供商申请新的 USIM 卡。
  • PIN 码和 PUK 码由您的 SIM/USIM 卡提供商提供，如需获取请联系 SIM/USIM 卡提供商。

操作步骤

  1. 选择“ > > ”。
  2. 从“”中，选择“”开启 PIN 码验证。
  3. 在“”中，输入正确的 PIN 码。
  4. 单击“”。
