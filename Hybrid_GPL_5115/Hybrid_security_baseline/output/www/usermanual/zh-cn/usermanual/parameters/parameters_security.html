安全

  • PIN 码自动校验
    开启 PIN 码自动校验功能，“”将会记住您的 PIN 码，并在每次需要校验 PIN 码时自动帮您校验。
  • 防火墙
    因特网是一个开放的网络，连接了全球各地的计算机。连接到网络上的任意一台计算机随时都有可能遭受到来自网络的攻击。“” 提供了强大的网络防护功能，可以在您的家庭网络和因特网之间构筑一道防护屏障，确保您家中的计算机免受网络攻击。
  • LAN IP 地址过滤
    允许或禁止局域网中指定 IP 地址对应的计算机使用指定的互联网服务。
  • LAN MAC 地址过滤
    允许或禁止指定 MAC 地址对应的计算机连接到局域网。
  • URL 地址过滤
    为了防止家庭成员浏览不适合访问的网站，您可以通过配置 URL 过滤功能禁止或只允许局域网用户访问指定的网站。
  • 虚拟服务器
    如果您的局域网内有一台计算机需要向互联网用户提供 HTTP（Hypertext Transfer Protocol，超文本传输协议）、FTP（File Transfer Protocol，文件传输协议）等服务时，您需要配置一个虚拟服务器。
  • 特殊应用程序
    对于视频会议、网络电话、网络游戏等应用程序，在客户端向因特网服务器请求连接的同时，服务器也需要向客户端发起连接。但在默认情况下，因特网服务器向局域网客户端发起的连接请求时， “”相应的端口并没有打开，导致连接中断。 通过设置端口触发功能，当局域网中有此类请求时，应用程序向触发端口发起连接，会触发“”打开相应的端口，从而保证该程序的正常使用。
  • DMZ 设置
    启用 DMZ 主机功能，可将设置为 DMZ 主机的计算机完全暴露给因特网。比如，搭建服务器时，您可以将需要对外提供服务的计算机设置为 DMZ 主机，以实现在对因特网提供服务的同时，确保家庭内其它计算机的安全。
  • ALG 设置
    应用层网关的主要功能是对应用层的信息进行解析，从而保证应用层上通信的正确性。
  • UPnP 设置
    UPnP（Universal Plug and Play，通用即插即用）支持多种网络设备的即插即用和自动发现。启用 UPnP，则允许支持 UPnP 的设备自动加入网络，简化配置操作。
  • ACL 设置
    通过 ACL（Access Control List，访问控制列表），您可以分别定义局域网计算机可以使用的局域网服务或互联网服务。
