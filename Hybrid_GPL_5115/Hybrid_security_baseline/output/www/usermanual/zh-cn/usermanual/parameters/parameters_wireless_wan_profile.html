无线 WAN 配置文件

一个无线 WAN 配置文件由一组无线上网参数组成。正确设置这些参数，使“”能够通过无线网络（4G/3G/2G）连接到互联网。

    • “”：

      从互联网中自动获取 APN。

    • “”：

      输入互联网服务提供商提供的 APN。

  • “”：

    输入互联网服务提供商提供的拨号号码。

  • “”：

    输入互联网服务提供商提供的上网用户名。

  • “”：

    输入互联网服务提供商提供的上网密码。

    • “”：

      加密的验证方式。这种方式能够避免传送用户的真实密码，安全性较高。
