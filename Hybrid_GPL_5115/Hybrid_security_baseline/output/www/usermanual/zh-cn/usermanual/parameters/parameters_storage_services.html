存储服务

您可以将 USB 存储设备连接到“”，开启存储共享服务，使家庭网络中的多台计算机能通过局域网同时访问 USB 存储设备中的数据，还可以对共享的数据进行查看、修改、删除等操作。

参数说明

  • “”：

    启用 Samba 共享服务后，您可以通过访问网络共享目录的形式，安全便捷地访问 USB 存储设备中的数据。例如在浏览器地址栏中输入“\\192.168.0.1”（“192.168.0.1”是 “” 的局域网接口地址）。

    说明：

    Samba 共享服务通常在家庭网络内部使用。由于因特网中的网络设备可能不支持 Samba 协议，因特网上的其它电脑可能无法访问通过 Samba 服务共享的数据。

  • “”：

    启用 FTP 共享服务后，您可以通过访问 FTP 服务器的方式，安全便捷地访问 USB 存储设备中的数据。例如在浏览器地址栏中输入“ftp://192.168.0.1”。

  • “”：

  • “”：

  • “”：

    • “”：

  • “”：
