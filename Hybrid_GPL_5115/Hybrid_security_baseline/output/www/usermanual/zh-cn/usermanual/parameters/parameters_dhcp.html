DHCP

  • DHCP
    开启 DHCP（Dynamic Host Configuration Protocol，动态主机设置协议）服务器功能，您的“”可以动态地为连接到局域网中的计算机或无线终端设备分配 IPv4 地址。
  • IPv6 DHCP
    如果您连接到局域网中的计算机和无线终端设备支持 IPv6 地址，您可以设置 IPv6 DHCP 服务为其动态分配 IPv6 地址和获取 DNS 服务器地址。
  • SLAAC
    如果您开启了 IPv6 DHCP 服务器功能，SLAAC（Stateless Address Auto-Configuration，无状态地址自动配置）可以为您的计算机和无线终端设备自动设置 IPv6 参数。
  • 静态地址
    通过设置静态地址，您可以为指定的计算机分配一个固定的局域网 IPv4 地址。
