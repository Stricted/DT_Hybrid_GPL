高级设置

  • DDNS
    动态域名服务 DDNS（Dynamic Domain Name Server）是一种将动态 IP 地址映射到一个固定的域名解析服务上的系统。当启用 DDNS 后，“”会把您计算机的动态 IP 地址传送给 DDNS 服务器，DDNS 服务器负责将更新后的 IP 地址与域名相对应。这样，因特网上的用户可以通过固定的域名来访问您提供的服务器资源，而不用亲自追踪您计算机的 IP 地址。
  • SNTP
    网络时间协议主要用于同步因特网中的计算机时钟。启用后，“”将从因特网上的时间服务器获取标准的时间，确保 “”系统时间的准确性。
  • 存储服务
    您可以将 USB 存储设备连接到“”，开启存储共享服务，使家庭网络中的多台计算机能通过局域网同时访问 USB 存储设备中的数据，还可以对共享的数据进行查看、修改、删除等操作。
  • 静态路由
    静态路由是指由网络管理员手动配置的固定路由信息。设置了一条静态路由后，“” 将在路由表中记录该路由项，从源 IP 地址发送的信息将按指定的路径发送到目的 IP 地址。
