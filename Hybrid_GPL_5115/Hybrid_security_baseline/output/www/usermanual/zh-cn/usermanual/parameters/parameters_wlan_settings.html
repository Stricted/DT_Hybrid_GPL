WLAN 设置

您的智能手机或安装了无线网卡的计算机，可使用无线网络功能实现无线上网，摆脱网线的束缚。

参数说明

基本设置

  • “”：

    选择您需要设置的 SSID 编号。“”可支持多个 SSID。

  • “”：

    SSID（Service Set Identifier，服务集标识）是一个用来标识 WLAN （Wireless Local Area Networks，无线局域网）的名称，使无线终端设备在连接无线局域网时能通过 SSID 来确定要连接的 WLAN 信号是否由“”提供。为了加强 WLAN 的使用安全，请修改默认的 SSID 名称和密码。

  • “”：

    定义当前 SSID 编号最多可以连接 WLAN 用户的数量。

WLAN 安全设置

  • “”:

    进行安全加密后，您的无线网络将受到保护，需要输入无线网络密码才能访问。推荐使用“”这种混合安全模式。

  • “”:

    通过选择合适的加密模式，快速完成无线网络的加密配置。

    • “”:

      Advanced Encryption Standard（高级加密标准），安全性比 TKIP 好，推荐使用。

    • “”:

      Temporal Key Integrity Protocol（临时密钥完整性协议），这是一种旧的加密标准。

    • “”:

      同时启用 TKIP 和 AES 加密模式

  • “”:

    WLAN 的密码。

说明：

建议您采用“”混合安全模式，并采用“”加密模式。这样可以在保证无线网络工作效率的前提下，确保无线网络的高安全性。

高级设置

  • “”:

  • “”:

  • “”:

    类似于有线网络的虚拟局域网（VLAN），将所有的无线客户端设备之间完全隔离，使客户端只能访问指定网络。

  • “”:
